(bloque not null: si es 1 indica tiene que ser no nulo
| Nº | Columna        | Tipo         | NOT NULL | Valor por defecto | PK |
| -- | -------------- | ------------ | -------- | ----------------- | -- |
| 0  | id             | INTEGER      | NO (0)   | —                 | 1  |
| 1  | nombre         | VARCHAR(100) | SÍ (1)   | —                 | 0  |
| 2  | email          | VARCHAR(100) | NO (0)   | —                 | 0  |
| 3  | fecha_registro | DATE         | SÍ (1)   | —                 | 0  |

PK = Primary Key

NOT NULL = 1 significa obligatorio -> tiene que tener algo escrito

para ver la llave forania:

sqlite> .schema pedidos
CREATE TABLE pedidos (id INTEGER PRIMARY KEY, cliente_id INTEGER, fecha_pedido DATE NOT NULL, total DECIMAL(10,2), FOREIGN KEY(cliente_id) REFERENCES clientes(id));

Observa cómo las constraints mantienen integridad:

INSERT INTO clientes (nombre, email, fecha_registro)
VALUES (NULL, 'a@b.com', '2025-01-01'); (aquí se agrega un nombre null y dará error)

INSERT INTO clientes VALUES (1, 'Luis', 'luis@mail.com', '2025-01-01');
INSERT INTO clientes VALUES (2, 'Pedro', 'luis@mail.com', '2025-01-05'); aquí aparecerá error porque email es unique


aquí se observa que al darle valores a 'productos' no se permite valores duplicados a id por ser primary key y no acepta duplicados
INSERT INTO productos (id, nombre, precio) VALUES (1, 'Laptop', 1000);


INSERT INTO productos (id, nombre, precio) VALUES (1, 'Mouse', 10);



Tu tabla pedidos tiene esta FK:
FOREIGN KEY (cliente_id) REFERENCES clientes(id)
Si intentas crear un pedido para un cliente que NO existe:
INSERT INTO pedidos (id, cliente_id, fecha_pedido, total) VALUES (1, 999, '2025-01-10', 50);

dira : FOREIGN KEY constraint failed ósea error.No puedes registrar pedidos para un cliente inexistente.
aunque al inicio se añadió, sucede que SQLite no tiene activada la llave forania. entonces hay que activarla

PRAGMA foreign_keys = ON;

INSERT INTO pedidos (id, cliente_id, fecha_pedido, total) VALUES (1, 998, '2025-01-10', 50); ahora ya lanza error
